## Projects

### Expense Tracker (`expense_tracker.py`)
A GUI-based expense tracking application built with Python and tkinter.

**Features:**
- Add, edit, delete expenses
- Category management
- Filterable reports with pie charts
- PDF export

**Dependencies:**
```bash
pip install matplotlib reportlab
```

**Run:**
```bash
python expense_tracker.py
```
